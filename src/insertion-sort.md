# Insertion Sort

## 소개

\\(O(n^2)\\)중에서는 개념적으로는 선택정렬까지는 아니지만 버블정렬보다는 쉽지만, 사실 잘 쓰지 않는 정렬 기법이다. \\(1\\)번부터 \\(n\\)번의 단계가 있는데, `i`번째 단계에서는 1번부터 `i-1`번째원소까지 모두 정렬되어 있는 상황에서 1번부터 `i`번까지 정렬한다. \\(1\\)번부터 `i-1`번까지 정렬되어 있으니, 그냥 뒤에서부터 훑어보면서 이 숫자가 들어갈 곳을 찾으면 된다.

## 코드

```c++
void sortarr()
{
    for(int i=1; i<=n; i++){
        int key=arr[i];
        for(int j=i-1; j>=1; j--){
            if(key>arr[j]){
                arr[j+1]=key;
                break;
            }
            swap(arr[j], arr[j+1]);
        }
    }
}
```
